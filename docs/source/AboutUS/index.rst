Introduction
============

ProperJobGaming is a modded minecraft server network founded late 2015. Hosting all major and up to date modpacks which are suitable for a big multiplayer experience. The servers are monitored and hosted on a dedicated server optimized for minecraft.

---------------

What we stand for
=================

* Good Stability, Uptime and Performance
* Working (Grief) Protection
* Friendly Community and Staff-Team
* Natural Modpack Experience
* Major and latest Modpacks
* Around for Years
